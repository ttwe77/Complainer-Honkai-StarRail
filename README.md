## 项目简介

**说怪话** 是一个 Windows 托盘通知小工具，灵感源自星穹铁道的奇美拉“说怪话”。它会定时在桌面右下角弹出“说怪话”风格的提示语，陪伴你度过每一个工作时刻。

## 功能特色

- 托盘驻留，定时弹出“说怪话”风格的通知
- 通知内容可自定义，支持多条随机切换
- 支持自定义通知周期
- 支持右键菜单手动触发通知或退出
- 图标与通知风格致敬奇美拉

## 使用方法

1. **下载与运行**
   - 编译或下载发布版，安装指示解压文件。
   - 双击运行Complainer.exe，程序会在托盘区常驻。

2. **自定义通知内容**
   - 通知内容存放在 `notices/notices.json`，可用文本编辑器修改或添加更多“说怪话”。

3. **自定义通知周期**
   - 编辑程序目录下的 `config.ini`，修改 `interval_seconds`（单位：秒）。

4. **手动触发/退出**
   - 托盘图标右键菜单可手动发送通知或退出程序。

## 开发&构建

- 依赖：.NET 8，WinForms
- 支持 Windows 10/11
- 项目结构：
  - `icon.ico`：程序与托盘图标
  - `notices/`：存放所有通知内容的 json 文件
  - `config.ini`：配置周期与通知文件
